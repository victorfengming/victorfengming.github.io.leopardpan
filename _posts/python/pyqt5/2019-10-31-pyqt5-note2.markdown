# PyQt5的第二课

### 使用QtWidgets写第一个程序,基于pyqt5

- 使用纯代码写第一个PyQt5程序

```python
import sys
from PyQt5.QtWidgets import  QApplication, QWidget

if __name__ == '__main__':
    app = QApplication(sys.argv)
    w = QWidget()
    w.resize(400,300)
    w.move(200,300)
    w.setWindowTitle("刘金玉编程")
    w.show()
    sys.exit(app.exec())
```
