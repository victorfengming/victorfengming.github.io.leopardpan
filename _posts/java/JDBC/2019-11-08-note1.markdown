### JDBC 概念
JDBC  (Java DataBase Connectivity)
Java数据库连接技术的简称，提供连接各种常用数据库的能力

### JDBC的工作原理

![jdbc2server](/images/posts/java/JDBC/jdbc2.png)
```html
JDBC API
提供者：Sun公司
内容：供程序员调用的接口与类
DriverManager
提供者：Sun公司
作用：管理各种不同的JDBC驱动
 JDBC 驱动
提供者：数据库厂商
作用：负责连接各种不同的数据库
```
